Global Attributes

id & class

id and class are global attributes which can be used with any element.

id

  • makes an element unique
  • can only be used once
  • may be used with any element
<section id="preamble">

class

  • used to group similar elements
  • can be used multiple times
  • can use multiple values (separated by spaces)
  • may be used with any element
<section class="branches-of-gov highlighted">

Using id and class

Values should:

  • start with a letter or underscore
  • be lowercase
  • provide context

Be careful of overuse. If you can apply styles, scripts to an element to make the design, interaction work, do that first. If more detail is necessary to identify elements, then consider using id or class.
