Internet 101

Origins of the Internet

  • DARPA (Defense Advanced Research Projects Agency) needed:
    • shared mainframe resources
    • secure communication in a crisis situation (nuclear war)
  • Resulted in ARPANET, a four-node network unveiled in 1969
    • University of California, Los Angeles
    • Stanford Research Institute
    • University of California, Santa Barbara
    • University of Utah

Netiquette

Early networks followed a somewhat unwritten social protocol. From MIT's Computing Handbook:

Personal messages to other ARPANet subscribers ... are generally not considered harmful ... Sending electronic mail over the ARPANet for commercial profit or political purposes is both anti-social and illegal. By sending such messages, you can offend many people, and it is possible to get MIT in serious trouble...

Beyond ARPANET

  • Grew to 213 host computers by 1981
  • Led to the development of subnetworks (i.e. MILNet)
  • Evolved into NSFNet — the Internet's backbone — in 1985
  • ARPANET decommissioned in 1990
  • NSFNet decommissioned in 1995, restrictions on commercial traffic lifted

ARPANET Lessons

  • Standardized communication protocols
    • Transmission Control Protocol (TCP)
    • Internet Protocol (IP)
  • Applications
    • E-mail
    • File Transfer Protocol (FTP)

Transmission Control Protocol (TCP)

  1. Computer A breaks a file into packets, attaches metadata
  2. The packets are sent over a network to Computer B
  3. Computer A receives, reassembles and verifies the file is correct

What if a packet goes missing?

The receiving computer will refer to the metadata sent with the file, as if it were a checklist and request a re-send.

Computer B: Hey, I'm missing packet No. 615!

Computer A: Let me try again.

Computer B: Got it!

Internet Protocol

  • Determines the path across a network which packets travel
  • Provides each node on a network with a unique address

What's a node?

A node is anything connected to a network, in this case, the Internet.

  • server
  • laptop
  • mobile phone
  • coffee machine
  • house!

Every device in the world...

IP Addresses

  • Each node has an Internet Protocol address
  • Internet Assigned Numbers Authority (IANA) delegates blocks of addresses to five regional Internet registries
  • IPv4 offers 232 possible combinations of addresses
    • Example: 99.66.22.215
    • 4.2 billion combinations have been exhausted (except for Africa)
  • IPv6 offers 2128 combinations
    • Example: 2602:306:3421:6d70:157d:7725:4f7:b182
    • These have been in use since mid 2000s

What is your IP Address?

  • Find Me By IP
  • Identifies location as it relates to the Internet and physical geography

Domain Name System

  • A 'directory' for the Internet
  • Overseen by ICANN (Internet Corp. For Assigned Names & Numbers)
  • Each domain is mapped to an IP address
    • google.com = 173.194.37.102
    • umiami.edu = 129.171.32.100
  • A Domain Name Server routes requests to the correct IP address

The Internet in Action

traceroute is a command line tool which tells us:

  • a packet's path to its final destination
  • the delay between each 'hop' from node to node

Anatomy of a Domain Name

email.miami.edu

  • edu - top-level domain
  • umiami - domain
    • we can 'reserve' these for our own use
  • email - subdomain (optional)
    • May point to a different server, directory
  • Combined, these create a fully qualified domain name

Top-level domains: Original / Generic

  • .com (commercial)
  • .net (network)
  • .org (organization)
  • .edu (educational)
  • .gov (government)
  • .mil (military)

Top-level domains: Countries

  • Two-letter country code
    • .us (United States)
    • .ca (Canada)
    • .de (Germany)
  • Some nations may restrict access to its citizens
  • Others have cashed in on domain 'hacking' (i.e. .co, .ly, .me, .tv)

Top-level domains: New

In 2008, ICANN began accepting applications for new top-level domains.

Many are generic (i.e. .business, .club, .sport) and accessible to all.

Some are limited for use by the company which holds a trademark (i.e. .android, .barclays, .microsoft).

ICANN recognizes over a thousand custom domains, but not all have been released for use.

Who can acquire a domain name?

Anyone!

  • Obtained through a domain name registrar
  • Fees vary depending on the name, sales
  • Minimum one-year lease
  • No one 'owns' a domain

Domain Name Directory

whois is a command line tool which tells us:

  • who a domain name is registered to
  • contact information for the lessee
    • some registrars offer privacy protection
  • when the registration will expire
