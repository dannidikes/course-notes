CSS Anatomy

CSS Blocks

This is a CSS block:

selector { property: value; }

Selectors

The selector identifies the part of our HTML to be styled.

This can be:

  • a specific element (<p>)
  • an id (#preamble)
  • a class (.branches-of-gov)

Complex Selectors

There are ways of combining selectors to isolate elements based on how they relate to other elements.

  • universal
  • type
  • child
  • descendant
  • adjacent sibling
  • general sibling
  • attribute

Declaration

The part enclosed within the curly braces ({ }) is called a declaration, which is comprised of a property and value.

selector { property: value; }

We can specify multiple declarations for each selector. We just need to separate each on with a semicolon (;). 

selector {
  property: value;
  property2: value2;
}

Properties & Values

A property and its value are separated by a colon (:).

There are hundreds! of properties we can change.

Values vary based on the property.

Examples of CSS Properties

  • Color
    • color (text color)
    • background-color
  • Text
    • font-family
    • font-size
    • font-style
    • font-weight
    • line-height
    • text-align
    • text-decoration
    • text-transform
