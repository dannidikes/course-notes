Document Object Model

What is the Document Object Model (DOM)?

The DOM is what results when a browser processes HTML. It:

  • turns our document into a tree-like structure
  • applies (hidden) properties
  • establishes relationships among elements

The DOM is an application programming interface (API) — simply put, the way in which one can interact with HTML using CSS or JavaScript.
